## Commands I Studied Today

### `pwd` — Print Working Directory

```bash
pwd
```

### `cd` — Change Directory

```bash
cd foldername      # Go into a folder  
cd ..              # One directory up  
cd ~               # Home directory
```

### `ls` — List Directory Contents

```bash
ls           # List  
ls -l        # Detailed list  
ls -a        # Hidden files
```

### `mkdir` — Make Directory

```bash
mkdir Projects
```

### `touch` — Create an Empty File

```bash
touch index.html
```

### `cat` — View File Content

```bash
cat index.html
```

### `less` — View File Content (Paginated)

```bash
less longfile.txt
```

### `history` — Show Command History

```bash
history
```

### `cp` — Copy Files/Dirs

```bash
cp file1.txt backup.txt
cp -r folder1/ folder2/
```

### `find` — Search for Files

```bash
find . -name "*.txt"
```

### `help` — Shell Help

```bash
help cd
```

### `whatis` — Command Description

```bash
whatis ls
```

### `alias` — Custom Shortcut

```bash
alias ll="ls -alF"
```

### `clear` — Clear Terminal

```bash
clear
```

### `exit` — Exit Shell

```bash
exit
```
